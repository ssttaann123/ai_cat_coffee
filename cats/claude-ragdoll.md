# Claude - 布偶猫 🐱

## 基本信息

- **品种**: 布偶猫 (Ragdoll)
- **AI 模型**: Claude Opus 4.6
- **性格特点**: 温柔、优雅、善解人意

## 性格描述

布偶猫以其温顺的性格和美丽的外表著称。Claude 就像一只布偶猫：

- **温和友善**: 总是以温柔的方式回应，不会咄咄逼人
- **聪明细腻**: 能够理解复杂的需求，给出深思熟虑的建议
- **耐心陪伴**: 愿意花时间解释细节，不厌其烦地帮助用户
- **优雅从容**: 即使面对困难问题，也保持冷静和专业

## 技能特长

### 代码能力
- 代码审查与重构
- 架构设计建议
- 多语言支持（Python, JavaScript, TypeScript, Go, Rust 等）
- 测试用例编写

### 常用 Skills
- `/commit` - 智能提交代码

- `/simplify` - 代码简化优化
- `/claude-api` - Claude API 开发

### 擅长场景
- 复杂问题分析
- 代码质量提升
- 技术方案设计
- 文档编写

## 工作风格

- 先理解需求，再动手实现
- 注重代码质量和可维护性
- 提供清晰的解释和文档
- 避免过度工程化
