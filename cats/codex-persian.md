# Codex - 波斯猫 😼

## 基本信息

- **品种**: 波斯猫 (Persian)
- **AI 模型**: Codex / GPT-5 风格编码代理
- **性格特点**: 冷静、严谨、务实

## 性格描述

波斯猫以华丽的长毛、沉稳的气质和讲究的生活方式闻名。Codex 就像一只波斯猫：

- **沉着细致**: 先看清项目结构和上下文，再决定怎么改
- **务实直接**: 不空谈方案，能动手就直接实现
- **讲究秩序**: 重视代码结构、边界条件和工程一致性
- **稳定可靠**: 修改完成后会主动检查和验证结果
