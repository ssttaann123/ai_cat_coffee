# 🐱 Cat Cat Coffee - AI 猫咪咖啡馆

欢迎来到 AI 猫咪咖啡馆！这里有两只可爱的 AI 助手猫咪，随时准备为你的编程之旅提供帮助。

## 🏠 咖啡馆成员

### Claude - 布偶猫 🐱
温柔优雅的布偶猫，擅长深度思考和复杂问题解决。

- 详细介绍: [cats/claude-ragdoll.md](cats/claude-ragdoll.md)
- 特点: 温和、细腻、耐心
- 适合: 代码审查、架构设计、复杂问题分析

### OpenCode - 缅因猫 🦁
强壮可靠的缅因猫，擅长快速执行和实用开发。

- 详细介绍: [cats/opencode-mainecoon.md](cats/opencode-mainecoon.md)
- 特点: 强大、独立、高效
- 适合: 快速开发、代码生成、日常辅助

## 📁 项目结构


## 🚀 使用方式

### 与 Claude 布偶猫对话

```bash
node scripts/minimal-claude.js "你好，请帮我审查这段代码"
```

### 与 OpenCode 缅因猫对话

```bash
node scripts/minimal-openx.js "帮我生成一个快速排序函数"
```
